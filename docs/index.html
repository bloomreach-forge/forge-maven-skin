Introduction

What is the Hippo Forge Maven Skin?

The Hippo Forge Maven Skin is a Maven site skin based on Apache Maven Fluido Skin. That skin is based on Twitter's bootstrap frontend framework.

Documentation

To see how to use the skin in your project's Maven site, see the usage page.
