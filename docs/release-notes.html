Release Notes

3.2.1

Release Date: 30 June 2023

  • FORGE-447 Fixed "Fork me on Github" ribbon.
  • Updated references to Forge maven repositories to use https, and to use the bloomreach domain.

3.2.0

Release Date: 22 September 2022

  • Change logo, now with a yellow Bloomreach 'b'.
  • Change first menu link from 'documentation' to 'xmdocumentation.bloomreach.com'.

3.1.1

Release Date: 24 April 2019

  • Change the copyright statement at the bottom to have 'Bloomreach Inc.' per default.

3.1.0

Release Date: 15 March 2019

  • FORGE-281 Set the URL behind the logo at bloomreach-forge.github.io, instead of to github.com/bloomreach-forge, and make it configurable in the descriptor element <logoUrl>, see the usage page.
  • FORGE-281 Rename one of the two breadcrumb links from onehippo.org to documentatioon.bloomreach.com and make both links configurable in the descriptor elements <breadcrumbLink1> and <breadcrumbLink2>, see the usage page.

3.0.1

Release Date: 8 November 2018

  • Set the URL behind the logo at github.com/bloomreach-forge.

3.0.0

Release Date: 1 November 2018

FORGE-231 Rebranding from Hippo to Bloomreach

  • Logo change, plus adding a site icon link.
  • Rename Maven groupId from org.onehippo.forge.mavenskin to org.bloomreach.forge.mavenskin, see the usage page.
  • Rename custom configuration element from <hippoSkin> to <bloomreach>, see the usage page.
  • Changed copyright notices.

2.0.1

Release Date: 11 July 2017

  • Fix for <span> elements appearing in source blocks. This resulted from a conflict between 'prettyprint' and 'syntaxhighlighter' Javascripts, so now the skin no longer supports the sourceLineNumbersEnabled setting that 'prettyprint' uses.

2.0.0

Release Date: 21 June 2017

  • Base the skin on Maven Fluido Skin version 1.6, used to be 1.4. Much better on mobile!
  • Modify the "Fork me on GitHub" ribbon support, defaulting to right top green ribbon, configurable in site.xml by element <custom><hippoSkin><gitHub><projectId>.
  • Small styling improvements.
